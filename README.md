# 🚐 TravelTrucks
**TravelTrucks** is a premium camper rental web application designed for travelers who seek freedom and comfort on the road. The platform allows users to explore a diverse fleet of campers, filter them by technical specifications, view detailed amenities, and book their next adventure through an intuitive interface.

---

## 🚀 Key Features

**Key Features:**
* **Dynamic Catalog:** A comprehensive listing of available campers with seamless data fetching and optimized rendering.
* **Advanced Filtering & Search:** Filter campers by location and vehicle type (Van, Fully Integrated, Alcove)
* **Detailed Camper Profiles:** Full technical specifications, high-quality image galleries, and authentic user reviews.
* **Professional Image Gallery:** An interactive gallery powered by Swiper, featuring synchronized thumbnails and smooth transitions.
* **Booking System:** Integrated reservation form with real-time field validation to streamline the rental process.

---

## 🛠 Tech Stack

**The project is built using a modern frontend development stack:**

* **Framework:** Next.js 16 (App Router)
* **Language:** TypeScript
* **Data Fetching:** TanStack React Query v5 & Axios
* **Forms & Validation:** Formik + Yup
* **UI Components:** Swiper (Galleries), React Icons
* **Notifications:** React Hot Toast
* **Styling:** CSS Modules & Modern Normalize
* **Backend:** Ready-to-use API by GoIT

---

## 🏗 Architecture & Performance
The project implements robust proxy logic and Middleware to ensure maximum security:

* **SSR & Hydration:** Implements HydrationBoundary for server-side prefetching, ensuring instant page loads and superior SEO performance.
* **Custom Branding:** Includes a custom-designed Full-Page Loader featuring the unique "TT" brand identity with smooth CSS animations.
* **Optimized Loading:** Uses Suspense boundaries to handle asynchronous UI states and prevent layout shifts.

## ⚙️ Installation & Setup
To run the project locally, follow these steps:

1. **Clone the repository:**
 ```bash
 git clone https://github.com/Onix43/project-campers
 cd project-campers
```

2. **Install dependencies:**
``` Bash
npm install
```

3. **Run the application in development mode:**
```Bash
npm run dev
```

4. **Build for production:**
```Bash
npm run build
npm run start
```
